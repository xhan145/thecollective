# SIGNAL//FLOW Media Kit Assets

Included files:
- signal_flow_brand_media_kit.docx: Editable brand media kit.
- signal_flow_primary_logo_dark.png: Primary logo on dark background.
- signal_flow_primary_logo_light.png: Primary logo on light background.
- signal_flow_icon_transparent_light.png: Light icon for dark backgrounds.
- signal_flow_icon_transparent_dark.png: Dark icon for light backgrounds.
- signal_flow_icon_dark_bg.png: App/social-ready icon on black background.
- signal_flow_wordmark_dark.png / signal_flow_wordmark_light.png: Wordmark assets.
- signal_flow_social_avatar.png: Social profile avatar.
- signal_flow_logo_source_concept.svg: Simple concept vector for designer handoff.

Usage note: This is a v1 concept kit. Before public launch, have a designer rebuild the final logo as production vector art and run trademark/domain checks.
